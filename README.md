## 1.Fyrir hvað stendur skammstöfunin RAM? Útskýrðu hvað RAM er.
RAM er random access memory ram er notad til ad talvan vinnur hradar. svo ad hardi diskurinn er ekki alltaf ad vinna a sama tima
## 2. Fyrir hvað stendur skammstöfunin ROM? Útskýrðu hvað ROM er
ROM er Read only memory Rom er notad ef kveikt er a tolvu og allt hverfur ef talvan er slökkt á.
